Notes (portuguese)
------------------
- Se um jogador ficar sem dinheiro poderá fazer all-in recebendo no máximo o número de jogadores vezes a sua aposta.
- Depois de stand ou double não há mais jogadas para esse jogador.
- Uma ação por ronda.
- Cada jogador poderá no máximo pedir onze cartas.
- No caso de todos estourarem devolve-se as apostas efetuadas por cada jogador.
- Se algum desistir divide-se o resto.
- Se algum jogador desistir só com duas cartas e sem ter feito stand é lhe devolvida metade da aposta, quantia a qual é retirada do montante total.
