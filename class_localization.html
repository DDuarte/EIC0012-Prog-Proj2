Descrição detalhada

Multi-language support.

This game supports multiple languages. Each ones is loaded from binary files encoded in UTF-8 (to support special characters) Sentences or words in the language file should be held apart by '/0' (null charcter)

Our program named "BinaryLanguageMaker" is used to create the files.

Singleton since we need to use it all around the code To get a string of the current language use the macro GetStr() and pass an element of the Strings enum
